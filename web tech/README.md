# 🌐 Personal Portfolio Website

Welcome to my portfolio site! This is a hand-crafted web space that showcases my work, skills, and passion for web development. Built using **HTML**, **CSS**, and **JavaScript**, it highlights my journey into responsive design, interactive interfaces, and animation magic.

## 🚀 Features

- Responsive layout for desktop and mobile
- Interactive navigation with smooth scroll
- Scroll-triggered animations using `IntersectionObserver`
- Fluid typography powered by `clamp()`
- Engaging hover effects and transitions
- Lightweight, fast-loading design

## 🎯 Purpose

This site was created as a personal branding project—a creative hub to share my projects, experiments, and what I'm learning in real-time. It’s also a sandbox to test out advanced front-end techniques.

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Modern CSS functions & media queries
-Bootstrap
